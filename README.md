# KILLA FAMQ

Русскоязычный Discord-бот для администрирования сервера: правила, подсказки по slash-командам, модерация, предупреждения, объявления и опросы.

## Что умеет

- `/помощь` - список команд прямо в Discord.
- `/правила` и `/правила-настроить` - хранение и публикация правил сервера.
- `/очистить`, `/мут`, `/размут`, `/кик`, `/бан`, `/разбан`, `/замедлить` - базовая админка.
- `/предупредить`, `/предупреждения`, `/снять-предупреждения` - локальный журнал предупреждений.
- `/сказать`, `/объявление`, `/опрос`, `/сервер` - полезные команды для канала.
- `/розыгрыш` - создать розыгрыш в giveaway-канале с кнопкой участия и автоматическим выбором победителей.
- Логи сервера в канал с названием вроде `🕠┃логи`: команды, удаления/изменения сообщений, модерация, каналы, роли и голосовые переходы.
- `/прайс` и `/прайс-настроить` - просмотр, публикация и обновление актуальных расценок.
- Лоты в продажах получают кнопки `Актуально` / `Неактуально`, чтобы автор или админ мог быстро закрыть старое объявление.
- Система повышения и форма отчётов на повышение с кнопкой `Создать отчёт`.
- Заявки в семью/корпу создают приватный канал для заявителя, лидера и деп-лидера, а закрытие переносит канал в архив.

## Запуск

1. Установи зависимости:

   ```powershell
   npm.cmd install
   ```

2. Создай `.env` по примеру `.env.example`:

   ```env
   DISCORD_TOKEN=токен_бота
   DISCORD_CLIENT_ID=id_приложения
   DISCORD_GUILD_ID=id_тестового_сервера
   DISCORD_LOG_CHANNEL_ID=
   DISCORD_ENABLE_MEMBER_LOGS=true
   DISCORD_ENABLE_MESSAGE_CONTENT_LOGS=true
   ```

   `DISCORD_LOG_CHANNEL_ID` можно оставить пустым: бот сам найдёт канал, если в названии есть `логи`. Флаги `DISCORD_ENABLE_MEMBER_LOGS` и `DISCORD_ENABLE_MESSAGE_CONTENT_LOGS` включены для полного журнала; ставь `false` только если такие privileged intents выключены в Discord Developer Portal.

3. Чтобы поставить имя `KILLA FAMQ` и прикрепленную аватарку, выполни:

   ```powershell
   npm.cmd run profile
   ```

   Этот скрипт лучше запускать только когда реально меняешь профиль: Discord ограничивает частые смены имени и аватарки.

4. Получи ссылку приглашения:

   ```powershell
   npm.cmd run invite
   ```

   Открой ссылку, выбери свой сервер и добавь бота. Скрипт дает боту `Administrator`, потому что он задуман как полный админ сервера.

5. Зарегистрируй slash-команды:

   ```powershell
   npm.cmd run deploy
   ```

6. Запусти бота:

   ```powershell
   npm.cmd start
   ```

Для теста оставь `DISCORD_GUILD_ID`: команды обновятся почти сразу. Для глобальной публикации убери `DISCORD_GUILD_ID` и снова выполни `npm.cmd run deploy`; глобальные команды могут появляться не мгновенно.
